Re-Enrolment
============

This site was created to manage the re-enrolment of a private school, as well as the “time bank” of the teachers.

Note that the texts in the site are in Portuguese, hence the file encoding is iso-8859-1 (latin1) which better suits the language characters, otherwise every accented word would have to be written using html special character format.
